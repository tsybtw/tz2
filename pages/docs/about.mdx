## Опис проєкту

Marketplace API - це REST API для торгової платформи, що дозволяє управляти продуктами та категоріями товарів. Проєкт створений в рамках лабораторної роботи з вивчення специфікації OpenAPI та інструментів документування API.

### Основні можливості

- **Управління продуктами**: створення, читання, оновлення та видалення товарів
- **Управління категоріями**: організація товарів за категоріями
- **Фільтрація та пошук**: можливість фільтрувати продукти за ціною та категорією
- **JWT автентифікація**: безпечний доступ до API
- **Автоматично згенерований SDK**: клієнтські бібліотеки для різних мов програмування

### Архітектура

API побудований відповідно до принципів REST та використовує:

- **HTTP методи**: GET, POST, PUT, DELETE
- **JSON формат**: для обміну даними
- **JWT токени**: для автентифікації
- **Стандартні HTTP статус-коди**: для індикації результатів операцій

## Технології

### Backend API
- OpenAPI 3.0 специфікація
- RESTful архітектура
- JWT автентифікація
- JSON Schema валідація

### SDK та інтеграція
- TypeScript SDK (згенерований автоматично)
- Axios HTTP клієнт
- Підтримка різних мов програмування

### Документація
- Zudoku для генерації документації
- GitHub Pages для хостингу
- Інтерактивні приклади API
